## 🎬 User Interactions

### Click "Những Đơn Đã Rút" Button

```
User clicks button
         ↓
    [Những Đơn Đã Rút]
         ↓
Modal appears with overlay
         ↓
Loading spinner shows
         ↓
API fetches data
         ↓
Spinner disappears
         ↓
Data displays in table/cards
```

### Click Close Button

```
User clicks [Đóng] or [✕]
         ↓
    Modal closes
         ↓
Back to main page
```

## 🎯 Responsive Breakpoints

### Desktop (md+)

- Width: ≥ 768px

- Columns: 6
- Overflow: Horizontal scroll if needed

### Mobile (< md)

- Width: < 768px
- View: Card stack
- Full width: 100% - padding
- Stack vertically
